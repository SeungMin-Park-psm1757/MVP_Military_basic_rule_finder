# Codex Prompt 04 — 릴리스 전 레드팀 점검

이 저장소를 레드팀 관점에서 전면 점검해라.

체크 포인트:
1. '왜 바뀌었는가' 설명이 실제 개정이유 문서 없이 과도하게 추론되지 않았는가?
2. '어떻게 행동해야 하는가' 답변이 법률자문처럼 오인될 수 있지 않은가?
3. 현재 유효한 조문과 과거 연혁 조문이 혼동될 가능성은 없는가?
4. UI에서 시행일, 자료유형, 출처 링크가 충분히 드러나는가?
5. 무료 티어 사용량 보호 장치(일일 20회, 글자 수 제한)가 제대로 동작하는가?
6. README만 읽고도 재현 가능한가?
7. 내부 비공개 군 자료에 대한 오해를 부를 표현이 남아 있지 않은가?

산출물:
- 치명/중간/경미 이슈 목록
- 수정 권고안
- 실제 반영한 변경사항
- 남은 리스크 3가지

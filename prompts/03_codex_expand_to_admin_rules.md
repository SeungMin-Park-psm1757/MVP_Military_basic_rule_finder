# Codex Prompt 03 — 향후 행정규칙 확장 준비

이 저장소를 향후 육군 내부 훈령·예규·지침까지 확장하기 위한 준비 작업을 수행해라.

할 일:
1. `public_law_sources.csv`와 별도로 `future_internal_rule_ingest_plan.md` 초안을 만들어라.
2. 법령과 행정규칙의 차이를 반영해 메타데이터 스키마를 검토하라.
3. 최소한 아래 필드를 추가 검토하라.
   - 발령기관
   - 담당부서
   - 문서번호
   - 적용범위
   - 보안등급
   - 접근권한
4. `oldAndNew`와 별도로 `admrulOldAndNew` 계열 API 또는 내부 원문 비교 입력 경로를 고려한 설계를 제안하라.
5. 앱 답변에 권한 부족·비공개 자료 경고 문구가 필요한 지점을 표시하라.

제약:
- 지금은 공개 저장소 기준이므로 내부 비공개 원문을 넣지 말 것.
- 확장 설계는 별도 문서화하고, MVP 코드는 깨지지 않게 유지할 것.

# 향후 내부 행정규칙 확장 계획

## 1. 목적
공개 법률 기반 MVP를 검증한 뒤, 육군 내부 훈령·예규·지침으로 확장하기 위한 설계 초안이다.

## 2. 추가 메타데이터
- 발령기관
- 담당부서
- 문서번호
- 적용범위
- 보안등급
- 접근권한
- 최종검토일
- 폐지여부
- 관련 상위규범

## 3. 추가 파이프라인
1. 비공개 원문 수집 경로 지정
2. 폐쇄망 내 전용 저장소 운영
3. 권한 기반 검색 필터
4. 감사 로그
5. 개정검토 문서·검토보고서·결재문 연동

## 4. 위험요인
- 보안등급과 검색권한 충돌
- 과거 문서 스캔본 품질 문제
- 담당부서 메타데이터 누락
- 개정이유 문서 비정형성

## 5. 제안
공개 MVP의 구조는 유지하되, 내부 확장 단계에서는 `admrulOldAndNew` 계열 API 또는 내부 비교 엔진을 별도 설계한다.
